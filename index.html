Projects

Materials Creative Co.

Materials Creative Co. is a project website built with an Angualar JS front-end and Ruby on Rails back-end. Materials Creative Co. is a site where a user can view unique t-shirts, find t-shirts, and have the ability to upload their own to demonstrate their awesome t-shirt style!

Going Places

Going Places is a complete front to back Ruby on Rails application. As a group project, we had to learn how to work as a team and simulate working with actual clients.

KingTUT

KingTUT is an app that was built using the MEN Stack which is MongoDB, Express.js and Node.js. KingTUT is an app where a user can save video tutorials to help them stay organizied while learning new skills or just a place to keep their favorite videos.
